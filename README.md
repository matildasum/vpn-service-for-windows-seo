## FAQ

**Does this VPN keep logs?**  
It’s built around a **No-Logs policy**. The point is privacy, not tracking.

**Is there a Kill Switch?**  
Yes. When enabled, it blocks traffic if the VPN drops so you don’t leak.

**Will it slow my connection down?**  
Any VPN adds overhead, but this one aims for **high speed** and a **stable connection** with good routing and server selection.

**Can I choose server locations?**  
Yes. Use **Global Servers** to pick a region and change it anytime.
